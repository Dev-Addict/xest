## Description

[Xest](https://github.com/Dev-Addict/xest) is [Next](https://github.com/vercel/next.js)
with [Nest](https://github.com/nestjs/nest) custom server.

## Installation

```bash
# using npm
  $ npm install
# using yarn
  $ yarn install
```

## Running the app

```bash
# development
  # using npm
    $ npm run start
  # using yarn
    $ yarn start

# watch mode
  # using npm
    $ npm run start:dev
  # using yarn
    $ yarn start:dev

# production mode
  # using npm
    $ npm run start:prod
   # using yarn
    $ yarn start:prod
```

## Test

```bash
# unit tests
  # using npm
    $ npm run test
  # using yarn
    $ yarn test

# e2e tests
  # using npm
    $ npm run test:e2e
  # using yarn
    $ yarn test:e2e

# test coverage
  # using npm
    $ npm run test:cov
  # using yarn
    $ yarn test:cov
```
